<HpResponse /> - whenever you fetch a request using \\\\\\*\\\\\\*http-pro\\\\\\*\\\\\\* you will get Response which will have following properties.

1. all the properties of javascript [Response](https://developer.mozilla.org/en-US/docs/Web/API/Response) object.
2. ### `data`

- data will have is the response provided by server.<br/><br/>
  :::note
  By default http-pro assumes that it will get json data from server. if you want change this behaviour change [responseType](HttpOptions#responsetype) to desired value.
  :::

```ts
export interface HpResponse<ResponseData extends any> extends Response {
  data: ResponseData;
}
```
