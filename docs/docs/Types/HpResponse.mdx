- whenever fetch a request using **http-pro** you will get Response which will following typescript type.

```ts
export interface HpResponse<ResponseData extends any> extends Response {
  data: ResponseData;
}
```

- In short it says that **http-pro** will return javascript [Response](https://developer.mozilla.org/en-US/docs/Web/API/Response) object with one additional property.
- which is named `data` and it contians whatever response data we get from server.
