- `http-pro` provides support for **node.js(v.18)** out of the box. but if you are working with **node.js(v.16)** you need to install one more additional package.


<Tabs>
<TabItem value="npm" label="npm">

```bash
npm install http-pro @http-pro/isomorphic 
```

</TabItem>
<TabItem value="yarn" label="yarn">

```bash
yarn add http-pro @http-pro/isomorphic 
```

</TabItem>
</Tabs>
